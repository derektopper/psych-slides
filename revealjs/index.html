reveal.js

Presentation template using slidify and reveal.js

The basics

Slidify is an R package that compiles R-Markdown scripts into html slideshows. The conversion from R-Markdown to html is done via "frameworks." reveal.js is one such framework. This template serves to illustrate some useful features of both slidify and reveal.js.
